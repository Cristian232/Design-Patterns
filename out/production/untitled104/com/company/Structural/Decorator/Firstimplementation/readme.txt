

                    Decorator Design pattern


        Definition:
            Decorator wropas the object within itself and provides the same interface as the wrapped object

        Use:
            To enhance behaviour of existing object dynamically

        Example: How_to

            Step 1: Create a Component that defines the interface needed or already used by client
            Step 2: Create a concrete component that implements the project
            Step 3: Create a decorator that implements the comp and also reference the concrete comp
